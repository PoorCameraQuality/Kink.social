# Hero / landing sections masterclass

Read [reference.md](reference.md) for multiple hero patterns with full code, responsive notes, and what to avoid.

## Apply first

- **Goal**: communicate value in seconds—clear headline, tight supporting copy, obvious primary action; optional proof and visual.
- **Avoid**: centered white text on stock photo + purple gradient as the default; match pattern to brand and offer.
- **Markup**: use `aria-labelledby` on the hero heading; respect heading order and contrast.
- **Performance**: optimize hero media (format, size, LCP); do not block the first paint with huge assets.
