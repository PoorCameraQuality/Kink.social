# Content blocks masterclass

Read [reference.md](reference.md) for full block types, HTML/CSS patterns, spacing, responsive behavior, and anti-patterns.

## Apply first

- **Section shell**: semantic `<section>` with `aria-labelledby` on the heading; shared container and header pattern before custom body content.
- **Avoid**: the same three equal cards for every section—pick a block type that matches intent (stats, split, logos, FAQ, etc.).
- **Consistency**: align spacing and max-width with your token system and other masterclasses (web, navigation).
